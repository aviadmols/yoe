# Discovery Duo Images

This folder collects image assets exported from Figma for the Shopify
`page.discovery-duo-new` template.

Files are grouped by section so it is easier to choose the correct asset for
each Shopify image picker. See `manifest.json` for the source URL, Figma
variable name, frame, variant, section, and intended usage for every file.

## Folders

- `hero/` - background, product, and award imagery (16 files)
- `stats-banner/` - before/after and clinical result imagery (37 files)
- `two-formulas/` - formula product images and routine graphic (12 files)
- `formula-comparison/` - SynerGenius diagram, mechanism icons, and active icons (44 files)
- `stubborn-stats/` - standards image and certification icons (20 files)
- `image-collage/` - collage block images (7 files)
- `retailer-gallery/` - logos, feature image, and gallery images (14 files)
- `refillable/` - refillable diagram, certification icons, and callout icons (48 files)
- `buy-box/` - folder reserved for trust/product imagery (0 files)
- `reviews/` - folder reserved for review/certification icons (0 files)
- `_uncategorized/` - Figma assets that could not be safely assigned to one section (16 files)

## Usage

- Upload assets from the matching folder into the corresponding Shopify image picker.
- If assets need to be used directly by the theme, copy them into `assets` with a clear flat prefix, for example `dd-hero-...`, because Shopify theme assets are safest when kept flat.
